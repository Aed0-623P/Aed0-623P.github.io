Venta de repuestos y accesorios para vehículos

  • Front-End Developer Y UI/UX Designer
  • Noviembre - 2024
  • URL: Sitio Web

Basado en el desarrollo de otro programador en WordPress, se modernizó y adaptó el formato UI/UX para el usuario y el administrador, dado que el sitio original estaba desactualizado para los estándares actuales.

Se trabajó de manera de ingeniería inversa para entender el programa y los plugins previamente desarrollados, analizando las limitaciones y capacidades de esta infraestructura en WordPress. Se realizó, en un ambiente local, una descomposición por partes para presentar al cliente si las estimaba necesarias o no. A la vez, se presentó de una manera más intuitiva la interfaz para que el cliente pudiera subir contenido cuando lo necesitara. Después de presentar el proyecto de diseño, se realizaron los trabajos para hacer realidad la propuesta, tanto en WordPress con sus plugins como en la creación de contenido visual con herramientas como Adobe illustrator, Canvas, etc.

Tecnologías utilizadas: PHP, Wordpress, Canvas, Anaconda, XAMPP, MySQL, FileZilla, Apache, LocalWP.
